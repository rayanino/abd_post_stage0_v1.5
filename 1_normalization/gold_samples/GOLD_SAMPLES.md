## Transformations applied

- Running headers (book title + page number) **stripped**
- Footnotes **separated** from matn using `<hr width='95'>` separator
- Footnote reference markers `(N)` **stripped** from matn (only when matching footnote exists)
- HTML tags (font color, span, div) **unwrapped**
- Whitespace **normalized** (collapse runs, trim lines)
- Footnote section **classified** as numbered_parens/bare_number/unnumbered/none

## Transformations NOT applied (preserved verbatim)

- Author's spelling and diacritics
- Parenthetical content that is not a footnote reference
- Verse star markers (`*`) — source data, not Shamela artifacts
- Punctuation patterns and spacing
- Verse hemistich separators (…)
